# 적용 현황

이 문서는 [적용해야할거.md](/home/work/llama_young/gpu-quddpm-lite-benchmark/적용해야할거.md) 기준으로 현재 프로젝트에 실제로 반영된 범위를 정리한 문서입니다.

중요:

- 이 프로젝트는 논문 완전 재현이 아닙니다.
- 현재 상태는 **QuDDPM/MSQuDDPM 아이디어를 바탕으로 한 본선 대비용 lightweight sandbox** 입니다.

## 한눈에 보기

| 항목 | 상태 | 메모 |
|---|---|---|
| 1. 진짜 빠른 smoke/mini preset 수정 | 완료 | smoke/mini preset 경량화 및 verify 동적 계산 반영 |
| 2. prior-based generation evaluation 추가 | 완료 | reconstruction/generation 분리 평가 반영 |
| 3. CNR comparator 추가 | 완료 | `cnr` 모델, 별도 학습 branch, generation-only 평가 반영 |
| 4. cumulative depolarizing + forward corruption 기록 | 완료 | cumulative/single_beta, corruption logging, `match_corruption` calibration 반영 |
| 5. resource metric 정교화 | 완료 | total resource columns, summary, plot, parameter-efficiency artifacts 반영 |
| 6. README 업데이트 | 완료 | scope, fairness, commands, generation 구분, ancilla/report 설명 반영 |
| 2차 개선용 프롬프트 | 완료 | `independent_step_quddpm`, parameter-efficiency 표/플롯 반영 |
| 3차 개선용 프롬프트 | 완료 | `ancilla_toy` concept demo 및 post-selection metrics 반영 |

현재 기준 핵심 남은 한계:

- `ancilla_toy`는 generation benchmark가 아니라 reconstruction-oriented concept demo입니다.
- resource metric은 여전히 heuristic이며, transpiled hardware count가 아닙니다.
- `match_corruption`은 physical equivalence가 아니라 operational fairness calibration입니다.

## 실행으로 확인한 항목

다음 명령은 실제로 실행 확인했습니다.

### smoke

```bash
python main.py --preset smoke --results-dir results_smoke
python verify_results.py --results-dir results_smoke
```

확인 결과:

- `metrics.csv` 생성
- `summary_table.csv` 생성
- plot 생성
- verify 통과

### baseline smoke

```bash
python main.py --preset smoke --models quddpm_baseline --results-dir results_smoke_baseline
python verify_results.py --results-dir results_smoke_baseline
```

확인 결과:

- `quddpm_baseline` 실행 정상
- `forward_process_type=random_unitary` 기록 확인

### mini with CNR

```bash
python main.py --preset mini --models msquddpm t_msquddpm cnr --results-dir results_mini
python verify_results.py --results-dir results_mini
```

확인 결과:

- `model=cnr` row 존재
- `generation_mmd`, `generation_wasserstein`, `generation_nearest_fidelity_mean` 기록 확인
- `msquddpm`, `t_msquddpm`는 reconstruction / generation 분리 기록 확인

### cumulative depolarizing smoke

```bash
python main.py --preset smoke --depolarizing-mode cumulative --results-dir results_smoke_cumulative
python verify_results.py --results-dir results_smoke_cumulative
```

확인 결과:

- `noise_curve.csv`에 `alpha_bar` 존재
- `forward_process_type=depolarizing_cumulative` 기록 확인

### iterative generation smoke

```bash
python main.py --preset smoke --generation-sampling-mode iterative --prior-mode maximally_mixed_jitter --results-dir results_smoke_iterative
python verify_results.py --results-dir results_smoke_iterative
```

확인 결과:

- `generation_sampling_mode=iterative` 기록 확인
- `generation_prior_mode=maximally_mixed_jitter` 기록 확인
